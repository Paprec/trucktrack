# trucktrack

projet de lecture bluetooth

## Start

To start the code, you need to execute :

`make run`

And in a new terminal :

`curl http://localhost:9090/addr`

The serveur should return a JSON type message with the following MAC addresses :

`01:01:01:01:01:01`
`01:01:01:01:01:02`
`01:01:01:01:01:03`
